使用说明：

1. 直接将.apk文件下载至手机中，即可点击安装，允许存储权限即可使用。

2. 该新闻App的网络数据通过本人PC作为服务器进行加载，故测试时如果本人PC机未运行server程序，刷新信息会显示失败。

项目编译：

1. 请将PermissionTest项目拷贝至无中文路径下，使用Android Studio打开，即可进行相关操作。

2. RssParser项目，是运行在PC端的服务器，服务器的启动位于rssparse\src\rssparse的java文件，运行RssParser的main函数即可。
